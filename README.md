<h1 align="center">Astro Miami</h1>

Minimalistic photography site with masonry grid. Built with Astro and TailwindCSS.

[**Demo Websites**](https://astro-miami.netlify.app)

## Features

- TailwindCSS
- Masonry grid

## Installation

```
pnpm i
pnpm dev
```
